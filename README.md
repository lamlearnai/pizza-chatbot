## Setup

```bash
git clone https://github.com/lamlearnai/pizza-chatbot.git
cd pizza-chatbot
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
copy .env.example .env    # then fill in your AI21_API_KEY
streamlit run streamlit_app.py
